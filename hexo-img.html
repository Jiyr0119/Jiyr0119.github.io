Hexo博客中插入本地图片

发表于 2019-07-01 | 分类于 博客搭建 | 评论数：

方法：安装插件 hexo-asset-image

  • 将 hexo 根目录下_config.yml 配置文件中的 post_asset_folder:项改为 true

  • gitBash 中执行

  • npm install hexo-asset-image –save

  • 新建.md 文件的同名文件夹,将待插入图片放入文件夹，.md 文件和文件夹位置同级（_posts 目录下）。

  • 使用愉快！
